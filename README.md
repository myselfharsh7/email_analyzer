Title: EMAIL ANALYZER | EMAIL Security Tool

Project Overview

Email Analyzer is a powerful tool built using Python and Django that analyzes email headers, checks for attachments and URLs using VirusTotal, and 
verifies spoofing and DNS records of the domain and also have google geminie intergrated chatbot. This comprehensive solution enhances email security by providing 
detailed insights and protections against common email threats. 

Usage: Analyze an Email:

Go to the email analysis section and upload your email file.
Click the "Analyze" button to get a detailed report.
Check Domain Spoof and DNS Records:

Enter the domain name in the provided input field.
Click the "Check Domain" button to get the spoof and DNS record status.
